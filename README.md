# FoodieApp
Food delivery application made with React-native
Features
  Firebase Auth
  Paystack Webview
  Add to Cart
  Menu Screen


Dependencies
  React-Redux
  Vector Icons
  Firebase
  Gesture Handler
  Paystack Webview
  SafeArea Context
  React-Navigation
